# Анализ эффективности A/B-теста посадочной страницы туристического сервиса

- Результат проекта

## Описание проекта
Цель проекта — определить, какой из двух вариантов посадочной страницы туристического сервиса (A — контрольный, B — тестовый) обеспечивает более высокую эффективность 
по ключевым бизнес-метрикам: конверсии в покупку тура и среднему чеку.
В рамках анализа исследуется корректность проведения A/B-теста, поведение пользователей во времени, стабильность метрик и статистическая значимость различий между группами. 
На основе результатов формируется рекомендация для бизнеса по выбору оптимального варианта страницы.

___

## Этапы работы над проектом
#### 1. Предобработка и проверка корректности эксперимента
- Анализ структуры данных, проверка типов и диапазонов значений.
- Приведение временных данных к нужному формату.
- Проверка длительности тестирования в группах A и B.
- Поиск и исключение пользователей, попавших в обе группы.
- Проверка данных на пропуски и аномалии, подготовка набора для анализа.
#### 2. Первичный анализ групп
- Расчет основных показателей: количество посещений, покупок и общая выручка по группам.
- Оценка метрик верхнего уровня: конверсия и средний чек в каждой группе.
- Сравнение покупательской способности по типам туров через сводные таблицы и визуализации.
#### 3. Анализ динамики и стабилизации метрик
- Формирование ежедневных метрик: визиты, покупки, конверсии, средние чеки.
- Построение кумулятивных показателей и графиков их изменения во времени.
- Проверка стабилизации метрик и корректного завершения эксперимента.
#### 4. Статистический анализ результатов
- Формулировка статистических гипотез для сравнения конверсии и среднего чека.
- Проверка нормальности распределений (тест Шапиро–Уилка).
- Сравнение конверсий с помощью z-теста для пропорций.
- Сравнение среднего чека с помощью t-теста для независимых выборок.
- Построение 95%-доверительных интервалов для конверсий, среднего чека и разности метрик.
  ___

## Использованные инструменты
- pandas, numpy — обработка и анализ данных
- matplotlib, seaborn — визуализация результатов и динамики метрик
- statsmodels, scipy.stats — статистические тесты и доверительные интервалы
  ___
## Результат проекта  
- Статистически значимой разницы в конверсии между вариантами A и B не выявлено (p-value > 0.05).
- Средний чек в группе B оказался значимо выше, чем в группе A; доверительные интервалы не пересекаются.
- Кумулятивные метрики стабилизировались, что подтверждает корректность и завершенность эксперимента.
- По итогам анализа рекомендовано использовать вариант B, поскольку он увеличивает суммарную выручку при сопоставимой конверсии.

Проект охватывает весь аналитический цикл A/B-теста — от подготовки данных и исследовательского анализа до статистической верификации результатов и формулировки бизнес-рекомендаций.
